# Car Price Prediction using Machine Learning

## Objective
The objective of this project is to predict the selling price of used cars using Machine Learning.

## Dataset
Cardekho Used Car Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Random Forest Regressor

## Steps
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Encoding
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Model Saving

## Evaluation Metrics
- RMSE
- R² Score

## Output
The model predicts the selling price of a used car based on its features.

## How to Run
1. Install the required libraries using `requirements.txt`.
2. Open the project in Jupyter Notebook.
3. Run all the cells in order.
